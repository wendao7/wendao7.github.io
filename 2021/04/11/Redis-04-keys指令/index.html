Redis-04-keys指令
win Lv8
2021-04-11 23:55:03
  • Redis
  • keys

1 KEYS

  • keys pattern

    • 作用：扫出指定模式的key列表

    • 缺点：Redis是单线程的，keys指令会导致线程阻塞一段时间，线上服务会停顿，直到指令执行完毕

      2 SCAN

  • SCAN cursor [MATCH pattern] [COUNT count]
    • 增量式迭代数据库中的键，每次执行只返回少量元素，不会阻塞服务器
    • 在对键进行增量式迭代的过程中， 键可能会被修改， 所以增量式迭代命令只能对被返回的元素提供有限的保证
      • 同一个元素可能会被返回多次。 处理重复元素的工作交由应用程序负责
      • 如果一个元素是在迭代过程中被添加到数据集的， 又或者是在迭代过程中从数据集中被删除的， 那么这个元素可能会被返回， 也可能不会， 这是未定义的（undefined）
