1 count

1.1 效果

  • count(*): 包含所有列，不忽略值为NULL
  • count(1): 包含所有列，不忽略值为NULL
  • count(列名): 只包含一列，忽略值为NULL

    1.2 效率

  • 列名为主键：count(列名)快于count(1)
  • 列名不为主键：count(1)快于count(列名)
  • 多个列且没有主键：count(1)快于count(*)
  • 只有一个字段：count(*)最优

2 exists和in的区别

2.1 exists

对外表用loop逐条查询，exists的条件就像一个bool条件，当能返回结果集则为true，不能则为false

2.2 in

多个or条件的叠加

2.3 使用

子查询表小的用in，否则用exists

3 union和union all的区别

将两个结果集合并为一个，两个结果集 字段个数必须一致，类型一致

  • union
    • 筛选掉重复的数据记录
    • 按字段顺序排序
  • union all
    • 不会去掉重复的数据记录
    • 不会对结果进行排序
